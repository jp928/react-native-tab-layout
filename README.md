# react-native-tab-layout

A react native library to link native TabLayout

## Installation

```sh
npm install react-native-tab-layout
```

## Usage

```js
import { TabLayoutView } from "react-native-tab-layout";

// ...

<TabLayoutView color="tomato" />
```
